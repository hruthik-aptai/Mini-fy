# BOOTSTRAP.md

One-time setup:

1. Decide whether this repo is a public template or your real private workspace.
2. Personalize `USER.md`, `TOOLS.md`, and `HEARTBEAT.md`.
3. Merge the relevant example from `config/` into `~/.openclaw/openclaw.json`.
4. Start a new session or restart the gateway, then run `openclaw skills list`.
5. Delete this file after setup to save recurring tokens.
