# Source Matrix

Use this skill when multiple sources need to be compared or reconciled.

## Workflow

1. List the claim or question being checked.
2. Gather the most relevant primary sources first.
3. Record source URL, date, scope, and the key supported fact for each.
4. Call out conflicts, gaps, or stale evidence.
5. Conclude with facts, then inference, then open questions.

## Guardrails

- Do not bury the date or source authority.
- Do not treat the newest source as automatically the best without checking scope.
- Do not collapse conflicting evidence into a false single answer.
